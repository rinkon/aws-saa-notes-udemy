### Section 4: IAM (Identity & Access Management)

- Users and Groups has policies attached as list of permissions
- Follow **Least Previlege Principle**
- Policies follow inheritance
- IAM - Password Policy
- Multi-Factor Authentication through various app and hardwares

Three ways to access AWS:
1. Management Console 
2. AWS CLI
3. AWS SDK

##### About Access Key
- Generated through management console.
- Access Key ID equivalent to username
- Secret Access Key equivalent to password
